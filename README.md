# Downloader-Desktop
[![Release](https://img.shields.io/github/v/release/sagin-pl/Downloader-Dekstop)](https://github.com/sagin-pl/Downloader-Desktop/releases)
[![Project](https://img.shields.io/badge/project-SAGIN--PL-green)](https://github.com/sagin-pl)

Downloader to aplikacja na różne platformy umożliwiająca pobieranie zdjęć i filmów z popularnych platform takich jak Instagram, TikTok, Youtube.

# Wymagania
- Komputer z systemem operacyjnym Windows (zalecane 10)
- Stałe, najlepiej WYŚMIENITE połączenie z Internetem

# Funkcje
- Pobieranie filmu
    - Z YouTube w NAJLEPSZEJ JAKOŚCI, lub HD
    - Z Instagram, TikTok w ORYGINALNEJ JAKOŚCI
- Zapis filmu do DOWOLNEJ LOKALIZACJI 
- Możliwość ODTWARZANIA filmu podczas pobierania </br>

# Instalacja
Aby zainstalować aplikację najpierw trzeba pobrać plik z zakładki [RELEASE](https://github.com/sagin-pl/Downloader-Desktop/releases).
Następnie należy otworzyć plik z rozszerzeniem .exe o nazwie downloader, bądź skrót o tej samej nazwie.
